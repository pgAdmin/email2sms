# Usage Instructions
### General Mode
 ```
 Forward your mails to <10-digit number>@smsmafia.in
 Ex: 98595598595@smsmafia.in
 The SMS will be delivered on this number between 9 AM and 9 PM (only if it is Non-DND)
 ```
### Special Mode
```
Create a way2sms.com account and Forward your mails to <username>_<password>@smsmafia.in
Ex: 98595598595_thepassword@smsmafia.in
The SMS will be delivered almost instantly and round the clock (subject to service availability of Way2sms.com).
```
